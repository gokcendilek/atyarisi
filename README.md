# At Yarisi Mini Oyun

Bu proje, Windows Forms uzerinde gelistirilmis basit bir **at yarisi simulatorudur**.
Her atin hizi ayarlanabilir, yaris ilerlemesi izlenir, kazananlar kaydedilir ve rapor disari aktarilabilir.

## Ozellikler
- 4 atli yaris simulasyonu
- At hizlari (max adim) ayarlanabilir
- Animasyon hizi ayarlanabilir
- Hava durumu ve sans etkisi
- Lider vurgusu, durum rozeti ve yaris ilerleme cubugu
- Son kazananlar ve yaris gecmisi
- Kazanan sayaci ve grafik gosterimi
- At isimleri ozellestirilebilir ve kaydedilir
- Tema degistirme (acik/koyu)
- Rapor disari aktarma (txt)

## Gereksinimler
- Windows 10/11
- .NET Framework (WinForms) destekli
- Visual Studio (onerilen) veya VS Code + .NET SDK (derleme icin)

> Not: WinForms uygulamalari **macOS/Linux** uzerinde dogrudan calismaz.

## Calistirma (Windows)
1. Projeyi indirin veya klonlayin.
2. `atyarisi.sln` dosyasini Visual Studio ile acin.
3. **Build > Run** ile calistirin.

## Kullanim
- **Baslat**: Yarisi baslatir.
- **Duraklat**: Yarisi durdurur.
- **Sifirla**: Yarisi sifirlar.
- **Rastgele Isim**: At isimlerini otomatik atar.
- **Rapor Disari Aktar**: Yaris raporunu `.txt` olarak kaydeder.
